# SSH Log Monitoring & Threat Detection using Splunk

## 📌 Overview
This project demonstrates SOC L1 level log analysis using Splunk to monitor SSH activity and detect potential security threats.

## 🎯 Objectives
- Detect failed and successful SSH login attempts
- Identify brute-force attacks
- Monitor unauthenticated connections
- Create dashboards and alerts

## 🛠️ Tools Used
- Splunk (SIEM)
- Linux SSH Logs

## 🔍 Use Cases
- Failed Login Detection
- Brute-force Attack Identification
- Suspicious Connection Monitoring
- Alert Configuration
